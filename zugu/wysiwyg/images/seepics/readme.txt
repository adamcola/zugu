See links is a file management program by elis.k.cage. Links appear automaticly 
followed by icons to [delete] [upload] and go [back] to previous pages.


--------------------------------------------------------------------------------

Everything is setup for you, so to plug and play simply;

1. Ftp Upload the file 'seelinks' to the folder you want to control.
2. make sure your set chmod '777' to that folder and the button "apply to subfolders and files" is ticked.
3. Set the file 'Seelinks' (and contents again!) to chmod of '155'.

What will happen if you DONT!

1. You will not be able to manage (i.e: upload/delete/link) The files make these things possible!
2. Otherwise when you click on delete your file will stay there because permission to delete is 'off'
3. This will stop the 'seelinks' folder from an accidental delete.
